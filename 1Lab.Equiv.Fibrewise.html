Fibrewise Equivalencesüîó

In HoTT, a type family P : A ‚Üí Type can be seen as a fibration with total space Œ£ P, with the fibration being the projection fst. Because of this, a function with type (X : _) ‚Üí P x ‚Üí Q x can be referred as a fibrewise map.

A function like this can be lifted to a function on total spaces:

total : ((x : A) ‚Üí P x ‚Üí Q x)
      ‚Üí Œ£ P ‚Üí Œ£ Q
total f (x , y) = x , f x y

Furthermore, the fibres of total f correspond to fibres of f, in the following manner:

total-fibers : {f : (x : A) ‚Üí P x ‚Üí Q x} {x : A} {v : Q x}
             ‚Üí Iso (fibre (f x) v) (fibre (total f) (x , v))
total-fibers {A = A} {P = P} {Q = Q} {f = f} {x = x} {v = v} = the-iso where
  open isIso

  to : {x : A} {v : Q x} ‚Üí fibre (f x) v ‚Üí fibre (total f) (x , v)
  to (v' , p) = (_ , v') , Œª i ‚Üí _ , p i

  from : {x : A} {v : Q x} ‚Üí fibre (total f) (x , v) ‚Üí fibre (f x) v
  from ((x , v) , p) =
    J (Œª { (x , v) _ ‚Üí fibre (f x) v } )
      (v , refl)
      p
  
  the-iso : {x : A} {v : Q x} ‚Üí Iso (fibre (f x) v) (fibre (total f) (x , v))
  the-iso .fst = to
  the-iso .snd .isIso.inv = from
  the-iso .snd .isIso.rinv ((x , v) , p) =
    J (Œª { _ p ‚Üí to (from ((x , v) , p)) ‚â° ((x , v) , p) })
      (ap to (JRefl {A = Œ£ Q} (Œª { (x , v) _ ‚Üí fibre (f x) v } ) (v , refl)))
      p
  the-iso .snd .isIso.linv (v , p) =
    J (Œª { _ p ‚Üí from (to (v , p)) ‚â° (v , p) })
      (JRefl {A = Œ£ Q} (Œª { (x , v) _ ‚Üí fibre (f x) v } ) (v , refl))
      p

From this, we immediately get that a fibrewise transformation is an equivalence iff. it induces an equivalence of total spaces by total.

total‚Üíequiv : {f : (x : A) ‚Üí P x ‚Üí Q x}
            ‚Üí isEquiv (total f)
            ‚Üí {x : A} ‚Üí isEquiv (f x)
isEquiv.isEqv (total‚Üíequiv eqv {x}) y =
  isHLevel-iso 0 (total-fibers .snd .isIso.inv)
                 (isIso.inverse (total-fibers .snd))
                 (eqv .isEqv (x , y))

equiv‚Üítotal : {f : (x : A) ‚Üí P x ‚Üí Q x}
            ‚Üí ({x : A} ‚Üí isEquiv (f x))
            ‚Üí isEquiv (total f)
isEquiv.isEqv (equiv‚Üítotal always-eqv) y =
  isHLevel-iso 0 (total-fibers .fst)
                 (total-fibers .snd)
                 (always-eqv .isEqv (y .snd))
