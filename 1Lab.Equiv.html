Equivalencesüîó

The big idea of homotopy type theory is that isomorphic types can be identified: the univalence axiom. However, the notion of isomorphism, is, in a sense, not ‚Äúcoherent‚Äù enough to be used in the definition. For that, we need a coherent definition of equivalence, where ‚Äúbeing an equivalence‚Äù is a proposition.

To be more specific, what we need for a notion of equivalence isEquiv(f)\mathrm{isEquiv}(f)isEquiv(f) to be ‚Äúcoherent‚Äù is:

  • Being an isomorphism implies being an equivalence (isIso(f)‚ÜíisEquiv(f)\mathrm{isIso}(f) \to \mathrm{isEquiv}(f)isIso(f)‚ÜíisEquiv(f))

  • Being an equivalence implies being an isomorphism (isEquiv(f)‚ÜíisIso(f)\mathrm{isEquiv}(f) \to \mathrm{isIso}(f)isEquiv(f)‚ÜíisIso(f)); Taken together with the first point we may summarise: ‚ÄúBeing an equivalence and being an isomorphism are logically equivalent.‚Äù

  • Most importantly, being an equivalence must be a proposition.

The notion we adopt is due to Voevodsky: An equivalence is one that has contractible fibres. Other definitions are possible (e.g.: bi-inverible maps) ‚Äî but contractible fibres are ‚Äúprivileged‚Äù in Cubical Agda because for glueing to work, we need a proof that equivalences have contractible fibres anyway.

A homotopy fibre, fibre or preimage of a function f at a point y : B is the collection of all elements of A that f maps to y. Since many choices of name are possible, we settle on the one that is shortest and most aesthetic: fibre.

fibre : (A ‚Üí B) ‚Üí B ‚Üí Type _
fibre f y = Œ£ Œª x ‚Üí f x ‚â° y

A function f is an equivalence if every one of its fibres is contractible - that is, for any element y in the range, there is exactly one element in the domain which f maps to y. Using set-theoretic language, f is an equivalence if the preimage of every element of the codomain is a singleton.

isEquiv is propositionalüîó

A function can be an equivalence in at most one way. This follows from propositions being closed under dependent products, and isContr being a proposition.

Isomorphisms from equivalencesüîó

For this section, we need a definition of isomorphism. This is the same as ever! An isomorphism is a function that has a two-sided inverse. We first define what it means for a function to invert another on the left and on the right:

isLeftInverse : (B ‚Üí A) ‚Üí (A ‚Üí B) ‚Üí Type _
isLeftInverse g f = (x : _) ‚Üí g (f x) ‚â° x

isRightInverse : (B ‚Üí A) ‚Üí (A ‚Üí B) ‚Üí Type _
isRightInverse g f = (x : _) ‚Üí f (g x) ‚â° x

A proof that a function fff is an isomorphism consists of a function ggg in the other direction, together with homotopies exhibiting ggg as a left- and right- inverse to fff.

  no-eta-equality
  constructor iso
  field
    inv : B ‚Üí A
    rinv : isRightInverse inv f
    linv : isLeftInverse inv f

  inverse : isIso inv
  inv inverse = f
  rinv inverse = linv
  linv inverse = rinv

Any function that is an equivalence is an isomorphism:

We can get an element of x from the proof that f is an equivalence - it‚Äôs the point of A mapped to y, which we get from centre of contraction for the fibres of f over y.

  eqv .isEqv y .centre .snd

Similarly, that one fibre gives us a proof that the function above is a right inverse to f.

  eqv .isEqv (f x) .paths (x , refl) i .fst

The proof that the function is a left inverse comes from the fibres of f over y being contractible. Since we have a fibre - namely, f maps x to f x by refl - we can get any other we want!

Equivalences from isomorphismsüîó

Any isomorphism can be upgraded into an equivalence, in a way that preserves the function fff, its inverse ggg, and the proof sss that ggg is a right inverse to fff. We can not preserve everything, though, as is usual when making something ‚Äúmore coherent‚Äù. Furthermore, if everything was preserved, isIso would be a proposition, and this is provably not the case.

The argument presented here is done directly in cubical style, but a less direct proof is also available, by showing that every isomorphism is a half-adjoint equivalence, and that half-adjoint equivalences have contractible fibres.
